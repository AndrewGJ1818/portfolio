# PROJECT REQUIREMENTS

## Functional Requirements

- The scroll animation must consist in a filing cabinet opening each of its drawers and folders.
- The website must include the following core sections — Home, About Me, Skills, Projects, and Contact.
- The main navigation menu must be accessible from all sections.
- Visual assets must be original and, if animated or interactive, must not interfere with usability or accessibility.
- The contact form must be fully functional.

## Non-Functional Requirements

- The website must be fully responsive and function correctly on desktops, tablets, and mobile devices.
- The color palette must have sufficient color contrast for text and UI elements.
- Semantic HTML structure must be used to support screen readers.
- All interactive elements must be fully navigable via keyboard.
- The website must have a Lighthouse accessibility score ≥ 90.
- The website must load within 2 seconds on a standard internet connection.
- All images must be optimized for web usage to minimize loading times.

## Constraints

- No JavaScript or frameworks — Vanilla HTML & CSS only.
- Must be hosted on GitHub Pages.
